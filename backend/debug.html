  • Read cookies from your default Chrome profile

  • Control tabs you opened manually

Your normal Chrome wasnt launched that way.

A brand-new Chrome VM, every time, unless you explicitly give it a saved profile.

How to fix your BrowserScraper

Step 1: Change browser launch logic

Inside browser.py, do this instead:

from playwright.async_api import async_playwright

class BrowserScraper:
    def __init__(self, platform: str, headless: bool = False):
        self.platform = platform
        self.headless = headless
        self.user_data_dir = "data/playwright_profile"

    async def _launch(self):
        self.pw = await async_playwright().start()

        self.context = await self.pw.chromium.launch_persistent_context(
            user_data_dir=self.user_data_dir,
            headless=self.headless,
            args=["--start-maximized"]
        )

        self.page = await self.context.new_page()
